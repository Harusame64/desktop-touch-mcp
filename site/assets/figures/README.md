# Figure Drafts

This directory stores rough visuals for the site.

## Files

- `hero-accident.svg`
  - two-panel hero visual for the home page
- `01-snapshot-vs-rpg.mmd`
  - the main comparison figure for the RPG article
- `02-rpg-four-ideas.mmd`
  - relationship map for provisional state, lens, guard, and lease
- `03-pre-action-flow.mmd`
  - pre-action validation flow
- `04-beyond-coordinate-roulette-map.mmd`
  - mindset map for the Beyond Coordinate Roulette page

## Usage note

- use Mermaid and SVG as fast roughs first
- tune color, spacing, and typography later when the real page layout exists
- keep text on the figure short and let the article do the explaining
