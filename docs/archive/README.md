# Archive Guide

`docs/archive/` は、履歴として残したいが日常運用では参照頻度の低い文書の置き場です。

## ルール

- 現行仕様ではなくなった計画書は `completed-plans/` に移す
- ローカルバックアップや退避ファイルは `backups/` に移す
- ルートの `docs/` に戻すのは、その文書が再び正本になったときだけ

## 今回移動した代表例

- `browser-eval-iife-roadmap.md`
- `mcp-ux-review-research.md`
- `rust-uia-engine-design.md`
- `rust-uia-engine-plan.md`
- `auto-perception-*`
- `background-input-plan.md`
- `v0.12-v0.13-trial-plan.md`

方針としては、チェックリストが閉じている文書や、すでに後続文書に役割を渡した計画書を優先的に移しています。
